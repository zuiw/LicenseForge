# LicenseForge

原创开源许可证合集。这是一个持续扩充的仓库，按法律特性分类收录自编的开源许可证。

## 分类导航

- **[copyleft/](./copyleft/)** — 强 copyleft / 保护性许可证。要求修改和衍生作品以相同许可证公开源码。
- **[permissive/](./permissive/)** — 宽松许可证。对使用、修改和再分发限制极少。
- **[special/](./special/)** — 特殊用途许可证。包含公有领域、源码可用、伦理、时间限制、非商业等特殊条款的许可证。

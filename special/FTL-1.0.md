## Preamble

The FTL addresses a critical gap in AI licensing: what happens when you
fine-tune a model?  Are LoRA weights a derivative work?  What about
partially fine-tuned checkpoints?  This license provides clear rules:
full fine-tunes are covered works, lightweight adapters (below a parameter
threshold) are not, and fine-tuned model outputs carry no license
obligations.

## TERMS AND CONDITIONS

### 0. Definitions.

"This License" refers to version 1.0 of the Fine-Tune License (FTL-1.0).

"The Base Model" means the model licensed under this License.

"Adapter Weights" means a set of parameters that are applied to a Base
Model without modifying the Base Model's original weights, including LoRA,
DoRA, AdaLoRA, and similar parameter-efficient fine-tuning methods.

"Full Fine-Tune" means any training process that modifies the Base Model's
original weights, or that produces Adapter Weights whose total parameter
count exceeds 10% of the Base Model's total parameter count.

"Lightweight Adapter" means Adapter Weights whose total parameter count
does not exceed 10% of the Base Model's total parameter count.

### 1. Base Model Rights.

You may use, copy, and distribute the Base Model under the terms specified

### 2. Lightweight Adapters.

Lightweight Adapters are not considered derivative works of the Base Model.
You may create, distribute, and use Lightweight Adapters for any purpose
without any license obligations under this License.

### 3. Full Fine-Tunes.

If you create a Full Fine-Tune and distribute it or use it in production,
you must:
- **a)** License the Full Fine-Tune under this License;
- **b)** Provide the Full Fine-Tune weights (or the delta weights);
- **c)** Document the fine-tuning process (dataset, hyperparameters,
  methodology);
- **d)** Clearly label the resulting model as a fine-tune of the Base Model.

### 4. Fine-Tune Outputs.

Model outputs generated by a Fine-Tune (whether Full or Lightweight) are
not subject to this License.  Users of the fine-tuned model own their
outputs.

### 5. Combined Models.

If you apply multiple Adapter Weights to the Base Model simultaneously,
the combination is not a derivative work unless the combined Adapter
Weights exceed the Full Fine-Tune threshold.

A 60-day cure period applies.

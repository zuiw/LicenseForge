The DCL prevents the software from being captured by centralized cloud
providers.  It prohibits offering the software as a hosted service on
infrastructure you control, unless you also provide users with the ability
to run their own independent instances with full data portability.  It is
designed for federated and peer-to-peer software that aims to resist
platform centralization.

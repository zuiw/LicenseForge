# Threshold Copyleft License, Version 1.0 (THRL-1.0)

**Copyright (C) 2026 [Your Name or Organization]**

A copyleft license that activates disclosure obligations only when changes exceed a threshold.

## Preamble

The THRL recognizes that not all uses of open source create derivative works of equal significance. Small modifications, bug fixes, and minor adaptations are excluded from copyleft obligations. Only when the cumulative changes exceed 30% of the original codebase do full copyleft obligations apply. This encourages adoption while still ensuring that significant improvements are shared back.

## TERMS AND CONDITIONS

### 0. Definitions.

"Modification Percentage" means (lines of code changed, added, or deleted) / (total lines of the original Licensed Work), measured at the time of distribution.

"Threshold" means 30%.

### 1. Permissions.

You may use, copy, modify, and distribute the Licensed Work, subject to the threshold conditions below.

### 2. Below Threshold.

If the Modification Percentage is below the Threshold, you may distribute the modified work under any terms, without source disclosure.

### 3. Above Threshold.

If the Modification Percentage equals or exceeds the Threshold, all copyleft obligations apply: source code must be disclosed and the derivative work must be licensed under THRL-1.0.

### 4. Measurement.

The Modification Percentage is calculated over the entire derivative work, aggregated across all modifications since the original version.

### 5. Termination.

Distributing above-Threshold modifications without disclosure terminates rights. A 60-day cure period applies.

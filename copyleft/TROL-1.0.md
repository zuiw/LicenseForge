## Preamble

The TROL combines copyleft disclosure with reproducible build requirements. Distributors must provide not just source code, but a verifiable, bit-reproducible build environment so users can independently verify that the distributed binary matches the source. This prevents supply chain attacks and backdoors.

"Build Environment" means all tools, scripts, configuration, and dependencies required to build the Software.

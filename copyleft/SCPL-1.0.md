## Preamble

The SCPL bridges AGPL and LGPL: it requires full source disclosure when the Software is used over a network, but explicitly permits linking from proprietary applications without triggering copyleft on the linking application. This makes it suitable for libraries and frameworks that want to ensure network-accessible modifications are shared while not restricting usage in proprietary software.

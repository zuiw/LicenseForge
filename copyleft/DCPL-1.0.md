## Preamble

The DCPL creates a temporal boundary: during active development, all contributors' changes must be shared under copyleft terms. Once a formal release is published, that release and all prior versions automatically convert to a permissive (MIT-equivalent) license. This encourages open development while reducing long-term licensing friction.

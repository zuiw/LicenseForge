## Preamble

The DDL closes the "AGPL loophole" where software used internally or behind a firewall escapes disclosure obligations. Under DDL, deployment itself—whether to production, staging, or any environment accessible by users—triggers source disclosure. If the software runs, the source must be available.

"End User" means any person or system that interacts with a Deployment.

### 2. Deployment Disclosure.

Any Deployment of the Licensed Work or its derivatives triggers the obligation to make Complete Corresponding Source available to all End Users.

Source code must be published on a publicly accessible repository and updated within 24 hours of any Deployment change.

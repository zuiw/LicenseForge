## Preamble

The SUNL addresses the "permanent copyleft" concern. It provides strong copyleft protection for a fixed term (5 years from release), after which it automatically converts to an MIT-style permissive license. This gives the author a head start while the project is young, and eventually frees the code for maximum adoption.

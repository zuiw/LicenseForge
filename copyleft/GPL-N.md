This license extends the GPLv3 with an Affero-style network disclosure

## Preamble

The GPLv3's lack of a network interaction clause leaves a loophole: modified
versions can be run on servers without disclosing source code.  GPL-N closes
this loophole by requiring that anyone who runs a modified version over a
network must offer the Corresponding Source to all remote users, just as
the AGPLv3 does, while keeping the rest of the GPLv3 framework intact.

## Preamble

The LIBL provides strong copyleft protection for the licensed work itself, while explicitly permitting use as a library in both open-source and proprietary software through a linking exception. Inspired by the LGPL but simplified and with a broader linking exception.

Modifications to the Library itself must be distributed under LIBL-1.0 with full source disclosure.

### 3. Linking Exception.

Works that Link to the Library are not considered derivative works and need not be licensed under LIBL-1.0. They may be licensed under any terms.

### 4. Combined Works Exception.

A Combined Work (a work that contains the Library alongside other software) is not a derivative work for purposes of this license.

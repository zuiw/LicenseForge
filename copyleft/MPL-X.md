This license extends the MPL-style file-level copyleft with a network
disclosure clause, making it suitable for modern web services and API-driven
architectures while preserving the permissive inter-module flexibility.

## Preamble

MPL-X retains the MPL's core approach: modifications to covered files must
be disclosed, but larger works can combine covered and non-covered modules
under different licenses.  It adds a network interaction clause for modified
files, stronger patent provisions, and a clearer definition of "File" in the
context of modern software architectures.

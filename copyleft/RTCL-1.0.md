# Real-Time Copyleft License, Version 1.0 (RTCL-1.0)

**Copyright (C) 2026 [Your Name or Organization]**

A copyleft license requiring real-time source disclosure triggered by every transaction or API call.

## Preamble

The RTCL goes beyond AGPL's network interaction clause. Every API call, transaction, or interaction with the licensed work must trigger real-time source disclosure to the requesting party. Source code must be delivered synchronously with the response, not merely offered. This eliminates the "source code available upon request" delay common in standard copyleft compliance.

## TERMS AND CONDITIONS

### 0. Definitions.

"Interaction" means any API call, message, transaction, or data exchange with the Licensed Work or its derivative.

"Real-Time Source" means the exact source code version corresponding to the running instance, delivered within the same interaction context.

"Running Instance" means the specific deployment of the Licensed Work that is processing the Interaction.

### 1. Permissions.

You may use, copy, modify, and distribute the Licensed Work, subject to the real-time disclosure conditions below.

### 2. Real-Time Source Delivery.

Upon any Interaction with the Licensed Work, you must deliver Real-Time Source to the interacting party as part of the response payload or via an automatically provided link.

### 3. Version Matching.

The Real-Time Source must exactly match the Running Instance. No delays, no "approximately equivalent" code, no obfuscated source.

### 4. Derivative Works.

Derivative works must maintain the same real-time disclosure requirement.

### 5. Termination.

Failure to deliver Real-Time Source upon request terminates rights. No cure period applies.
